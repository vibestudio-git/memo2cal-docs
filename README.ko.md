## Memo2Cal 문서 허브 (한국어)

이 저장소는 Memo2Cal 크롬 확장프로그램의 공개 문서를 제공합니다.
빠른 메모를 Google 캘린더 일정으로 저장하고, 주기적 알림으로 메모 습관을 형성할 수 있습니다.

> 공식 사이트: https://memo2cal.vibestudio.live/

이 문서는 한국어입니다. English: [README.md](./README.md)

### 빠른 시작
- 사용자 매뉴얼: https://memo2cal.vibestudio.live/user-manual.html
- 자주 묻는 질문(FAQ): https://memo2cal.vibestudio.live/faq.html
- 개인정보처리방침: https://memo2cal.vibestudio.live/privacy-policy.html

### 주요 기능 하이라이트
1. 빠른 메모 → 캘린더 일정 생성
2. 주기 알림(30분/1시간 또는 시간 단위)
3. 다중 캘린더 선택
4. 오프라인 시 로컬 임시 저장
5. 업무 시간대/알림 토글
