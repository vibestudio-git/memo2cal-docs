Memo2Cal FAQ

Last updated: 2025-10-26

1. Google account linking

Q: I see “Google Calendar connection is required.”
A: Re‑link may be needed if you restarted Chrome, the extension was updated, or Google Account security settings changed. Click “Connect Account”.

Q: The Connect button does nothing.
A: Close/reopen the popup → restart Chrome → Manage extensions → disable/enable.

Q: Linked, but calendars/events won’t load.
A: Check Google Account permissions and ensure “Memo to Calendar” exists; if not, re‑link.

2. Event management

Q: Does it only show today?
A: The focus is “today” by default (00:00–23:59). Use the calendar icon to select another date.

Q: No scrollbar.
A: It appears automatically when there are 4 or more items.

3. Settings and features

Q: Change reminder interval?
A: Use Settings → 30 min / 1 hour or set hourly values. Set working hours; toggle OFF to disable reminders.

4. Troubleshooting

  1. Refresh the extension (close and reopen the popup)
  2. Re‑run linking (Connect Account)
  3. Restart the browser
  4. Re‑enable from Manage extensions

If it persists: open DevTools (F12) → Console to check errors, then email smartbigbang@gmail.com.

1. Google 계정 연동

Q: “구글 캘린더 연동이 필요합니다”가 표시됩니다.
A: 브라우저 재시작/확장 업데이트/계정 보안설정 변경 시 재연동이 필요할 수 있습니다. “계정연동”을 클릭하세요.

Q: 계정연동 버튼이 동작하지 않습니다.
A: 팝업 닫고 다시 열기 → 브라우저 재시작 → 확장프로그램 관리에서 비활성/재활성.

Q: 연동 후에도 캘린더/일정이 보이지 않습니다.
A: Google 계정 설정에서 “Memo to Calendar” 존재 확인 후, 없다면 재연동하세요.

2. 일정 관리

Q: 오늘 일정만 보이나요?
A: 기본은 “오늘(00:00–23:59)”입니다. 달력 아이콘으로 다른 날짜를 선택하세요.

Q: 스크롤이 보이지 않습니다.
A: 4개 이상이면 자동으로 스크롤바가 표시됩니다.

3. 설정 및 기능

Q: 알림 간격 변경?
A: 설정에서 30분/1시간 또는 시간 단위로 선택하세요. 업무 시간대를 지정하고, 알림 토글로 비활성화할 수 있습니다.

4. 문제 해결

  1. 팝업 닫았다가 다시 열기
  2. 계정연동 재시도
  3. 브라우저 재시작
  4. 확장프로그램 재활성화

지속 시 DevTools(F12) → Console 오류 확인 후 smartbigbang@gmail.com로 문의하세요.
